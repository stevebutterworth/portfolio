Portfolio lightbox — two takes

Opens when a project cover with extra media is clicked. Shows images + video, with a thumbnail strip, prev/next, counter, title and close. Keyboard: Esc closes, arrows navigate. A = dark immersive (images pop); B = light framed panel (on-brand ivory).
